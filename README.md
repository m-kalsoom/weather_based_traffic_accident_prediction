Weather based Traffic Accident Prediction

Weather-Based Traffic Accident Prediction using Python, Pandas, NumPy, seaborn and Scikit-Learn with missing value handling and feature engineering for machine learning applications. Data Analysis, Feature Engineering, Visualization, and Machine Learning to study the impact of weather and traffic conditions on road accidents.

## Project Objectives

* Analyze traffic and accident datasets.
* Explore the impact of weather conditions on accidents.
* Perform data cleaning and preprocessing.
* Handle missing values and inconsistent data.
* Engineer meaningful features for machine learning.
* Visualize accident trends and traffic patterns.
* Build predictive models for accident risk assessment.

## Installation

Clone the repository:

```bash
git clone https://github.com/m-kalsoom/weather_based_traffic_accident_prediction.git
```

Move into the project directory:

```bash
cd weather_based_traffic_accident_prediction
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```
